PROJECT NAME

Demise Message Builder PWA

OBJECTIVE

Build a production-ready Progressive Web App (PWA) for creating professional demise announcements and condolence messages commonly used in Banaskantha, Palanpuri and Gujarati community groups.

The application must be mobile-first, responsive, installable as a PWA, work on all screen sizes, and generate WhatsApp-ready messages with proper formatting.

The application is NOT a generic form builder.

It is a highly optimized demise-message drafting platform.

====================================================
