🔗 Developer & Spreadsheet Connection Setup

This Progressive Web App fetches templates, relationship defaults, and settings directly from your private Google Spreadsheet. It also uses Google Apps Script to proxy AI queries securely without exposing API keys.

  1. Create a Google Spreadsheet.
  2. Click Extensions → Apps Script.
  3. Paste the code from Code.gs into the editor.
  4. Run the setupSpreadsheet function once to create the worksheets automatically.
  5. Click Deploy → New Deployment, set access to "Anyone", and copy the Web App URL.
  6. Paste the Web App URL inside your PWA config in app.js or configure it directly in settings.
